# Project Plan - E-commerce KPI Dashboard

## 1. Project Goal
The goal of this project is to analyze customer behavior and build a KPI dashboard to support business decision-making.

## 2. Key KPIs
- Total Revenue
- Conversion Rate
- Average Order Value
- Returning vs New Visitors
- Purchase Rate

## 3. Dashboard Structure
- Overview (Revenue, Conversion Rate)
- Trends (Monthly performance)
- Customer Analysis (New vs Returning)
- Insights (Factors influencing purchase)

## 4. Tools
- Python (data cleaning & analysis)
- SQL (KPI calculation)
- Power BI (dashboard)

## 5. Next Steps
- Data exploration
- KPI calculation
- Build dashboard
